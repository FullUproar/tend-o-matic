# @tend-o-matic/ui

Touch-first design system. Thumb-zone components, fuel-gauge limit indicators, scan-first inputs. Shared across till, backoffice, and portal.

## Theme

Brand palette is drawn from the tend-o-matic poster: parchment ground, kraft browns, mustard, leaf green, clay terracotta, ink. Tokens live in `src/tokens.ts`; consume them either as TS constants or via the Tailwind preset (`src/tailwind-preset.ts`).

Application guidance lives in `src/tokens.ts` — short version: the till stays "ink on parchment" for legibility; deeper kraft / mustard / clay treatments belong to portal chrome, receipts, and marketing.
